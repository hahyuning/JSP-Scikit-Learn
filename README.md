# JSP-Scikit-Learn
텍스트 분석을 이용한 영화 추천 사이트

## 개요
1. 참여 인원: 3명
<br>

2. 사용 기술
+ Java
+ MySQL
+ JDBC
+ JSP
+ Ajax
+ JQuery
+ Python
+ Selenium
+ Scikit-Learn
<br>

3. 맡은 역할
+ 회원가입, 로그인, 계정 관리 기능
+ 크롤링을 이용한 개봉 예정인 영화 데이터 수집
+ 역대 흥행 영화 순위별 정보 출력
+ 텍스트 분석을 통한 유사 영화 추천 기능
<br>
